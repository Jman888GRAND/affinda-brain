# Fundraising

**Status:** 📄 CV only

## Definition
The ability to identify funding opportunities, make compelling cases for support, and secure financial or in-kind resources for a program or organisation.

## Evidence
- Managed fundraising activity as Head of Boys' Volleyball Program at Huntingtower School, securing resources to support a 30+ athlete program and a 35% participation increase over the program tenure
- Growing the program from 15th to 3rd at the National Schools Cup required sustained resource acquisition alongside coaching improvements
